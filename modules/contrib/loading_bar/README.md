### SUMMARY
This module provides a highly flexible progress bar form element based on SVG,
using [LoadingBar.js library](https://github.com/loadingio/loading-bar/).

### INSTALLATION
Using composer:
```
composer require drupal/loading_bar --sort-packages
```
